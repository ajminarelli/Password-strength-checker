 Password Strength Checker
A simple command line tool built with Python that checks the strength of a password and tells you exactly what's missing to make it stronger.

Features:

-Rates your password strength
-Tells you specifically what's missing, such as:

Too short — needs more length
Missing lowercase letters
Missing uppercase letters
Missing numbers
Missing special characters (e.g. !@#$%)



Built With:

Python

How to Use:

Run the script in your terminal
Enter a password when prompted
The app will rate its strength and list any improvements needed
